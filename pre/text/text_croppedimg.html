CROPPED IMAGES

 In this window you can see three different buttons.
  • Create cropped images:cuts the annotation and shows it in the panel above. It allows to save it in the pre/crop_imgfoldeer
  • Train the model:it trains the model based upon the annotation.
  • Make prediction:opens a new window in which we can:
    • choose the images on which we can do some predictions
    • use the buttons probability and category to choose how to display our prediction
