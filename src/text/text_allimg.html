In the 'All images' window there are all image that we have uploaded.
With the 'Open Images From Folder' button you can choose which folder you want to use to open the images. By default the program uses the 'img' folder in the project one.
To select an image you can click on it, however if you want to elect alls the images, you can click on the right button 'CheckAll'.
To unselect an image, as for the selection, you can click on it.
To unselect all the images, you can click on the 'UncheckAll' button.
